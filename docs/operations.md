# Monthly Operations Checklist

## Week 1: Planning
- [ ] Review previous month KPIs
- [ ] Identify underperforming clients
- [ ] Plan content calendar
- [ ] Assign team tasks

## Week 2: Content Creation
- [ ] Generate 4-8 social media posts (AI)
- [ ] Write 2 blog posts per client
- [ ] Create 2-4 Google Business posts
- [ ] Update chatbot FAQs

## Week 3: Execution
- [ ] Publish all social content
- [ ] Update Google Business profiles
- [ ] Deploy chatbot changes
- [ ] Optimize landing pages

## Week 4: Reporting
- [ ] Generate monthly reports
- [ ] Compile KPIs
- [ ] Send to clients
- [ ] Review churn/renewal risk

## Critical Metrics

**Per Client:**
- Website visits
- Form submissions
- Chatbot conversations
- Google ranking positions
- Review count/rating

**Overall:**
- MRR (Monthly Recurring Revenue)
- Churn rate
- Upsell rate
- Cost per acquisition

## Tools Setup

```
- CRM: Notion (leads + clients)
- Hosting: Hostinger (sites)
- Email: SMTP (for reports)
- Scheduling: Calendly (for calls)
- AI: OpenAI (ChatGPT API)
- Billing: Stripe (subscriptions)
- Voice: Harpoon (outbound calls)
```
